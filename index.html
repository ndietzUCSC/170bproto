Prototype 2 - Norah Dietz

How can EATING, T-SHIRTS, create a sense of CONFUSION.

I chose to to learn Armory3D, because a realtime engine attached to Blender seems useful for grebox prototyping.
Of course, an open source engine comes with stability issues, which I repeatedly ran into. Otherwise, I found it very intutive as a visual programming based engine.

Build Info

Background

You run a pants store but are secretly selling unlicensed T-shirts. Uh oh... The inspectors just walked in.

Goal

Grab and eat the illicit T-shirts before the inspectors notice.
This is the only option. You must eat them.

Controls

Use A and D move the left or right hand. Release to grab an item.
Make sure to click the game window at lease once, to enable sound.
