Prototype 2 - Norah Dietz

How can EATING, T-SHIRTS, create a sense of CONFUSION.

I chose to learn Armory3D, because a real-time engine attached to Blender seems useful for greybox prototyping.
Of course, an open source engine comes with stability issues, which I repeatedly ran into. Otherwise, I found it very intuitive as a visual programming based engine.

Build Info

Background

You run a pants store but are secretly selling unlicensed T-shirts. Uh oh... The inspectors just walked in.

Goal

Grab and eat the illicit T-shirts before the inspectors notice.
This is the only option. You must eat them.

Controls

Use A and D to move the left or right hand. Release to grab an item.
Make sure to click the game window at least once to enable sound.

Known Bugs and Limitations

No scoring or fail states.
Mainly testing the hold and release control scheme.
Hands can get stuck/lose their correct origin.
Eating sfx will play repeatedly despite grabbed object disappearing.
